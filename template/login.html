Login Help

Having trouble signing in? Try these steps:

"Invalid Credentials" Error?

  • Ensure Caps Lock is off.
  • Click "Forgot Password?" to reset (check spam folder).

Page Freezes After Login?

  • Disable browser extensions (especially ad blockers).
  • Switch from Wi-Fi to mobile data or vice versa.

No Password Reset Email?

  • Wait 5 minutes, then search for emails from troubleshoot@swadesh.com.
  • Still missing? Contact Support with your registered email.

"Account Not Found"?

  • Ensure you're using the same email/phone number used to register.
